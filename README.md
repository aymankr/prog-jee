## Commandes à retenir

### Maven
 - mvn archetype:generate <nom> <version> ...
 - mvn compile
 - mvn test
 - mvn package
 - mvn clean
 - mvn site

### Java compilation
 - javac (.java --> .class)
 - java (.class)
 - jvm
